# Release Notes — v0.2.19

**Date:** 2025-04-10
**Repository:** nexus-llm-router

## Highlights

This release focuses on stability and developer-experience improvements
to the `router` core.

## Changes

### Added
- `provider` module: added configurable retry strategy with jitter
- New `--verbose` flag for detailed strategy trace output
- Unit tests for router boundary conditions

### Fixed
- Off-by-one in pagination logic when result set is empty
- Memory leak in long-running provider worker loop
- Incorrect timezone handling in audit timestamps

### Improved
- Reduced average strategy latency by ~12% via connection pooling
- Cleaner error messages for router validation failures

## Upgrade Notes

No breaking changes. Drop-in upgrade from the previous version.

```bash
pip install --upgrade nexus-llm-router
```
