# Release Notes — v0.2.3

**Date:** 2024-10-11
**Repository:** nexus-llm-router

## Highlights

This release focuses on stability and developer-experience improvements
to the `router` core.

## Changes

### Added
- `strategy` module: added configurable retry strategy with jitter
- New `--verbose` flag for detailed cost trace output
- Unit tests for adapter boundary conditions

### Fixed
- Off-by-one in pagination logic when result set is empty
- Memory leak in long-running strategy worker loop
- Incorrect timezone handling in audit timestamps

### Improved
- Reduced average cost latency by ~12% via connection pooling
- Cleaner error messages for adapter validation failures

## Upgrade Notes

No breaking changes. Drop-in upgrade from the previous version.

```bash
pip install --upgrade nexus-llm-router
```
